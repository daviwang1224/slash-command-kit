# 示例提示词

这是一个示例提示词文件，演示如何在 Cursor 和 Claude Code 中创建自定义斜杠命令。

## 使用方法

在编辑器中输入 `/example` 即可调用此命令。

## 提示词内容

你好！这是一个示例提示词。

你可以在这里编写任何你想要的提示词内容，包括：

- 代码生成模板
- 文档编写指南
- 代码审查清单
- 测试用例生成
- 重构建议
- 等等...

## 变量替换

你可以使用变量让提示词更加灵活：

- `$SELECTION` - 当前选中的文本
- `$FILE` - 当前文件路径
- `$LANGUAGE` - 当前文件的编程语言

示例：
```
请帮我重构以下代码：

$SELECTION

使用 $LANGUAGE 的最佳实践。
```
